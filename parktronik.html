  • Теория микроконтроллеров (10 мин)

  • Программирование на C++ (20 мин)
  • Отладка и тесты (10 мин)

02. Проблема

Статистика ДТП при парковке:

  • 30% аварий происходят задним ходом.
  • Причина: "Слепая зона" зеркал.
  • Решение: Акустический сенсор (Парктроник).

03. Наше решение

Мы создадим копию системы Tesla:

  1. Датчик замеряет дистанцию (см).
  2. Контроллер вычисляет степень угрозы.
  3. Система подает световые и звуковые сигналы.

04. Инструментарий

Среда разработки: Tinkercad Circuits.

Почему не реальное железо?

  • Мгновенная сборка.
  • Невозможно сжечь компоненты.
  • Встроенный осциллограф и монитор порта.

MODULE: THEORY

HARDWARE ARCHITECTURE

06. Arduino Uno

Центральный процессор (ATmega328P).

> Clock Speed: 16 MHz > Digital I/O: 14 Pins > Voltage: 5V

Это мозг, который выполняет наш код строка за строкой.

07. Макетная плата

Breadboard — устройство для прототипирования.

  • Шина питания (+/-): Вертикальные линии.
  • Терминальные полосы: Горизонтальные линии (A-E, F-J).

Внутри пластика скрыты металлические контакты.

08. Датчик HC-SR04

Принцип: SONAR (Sound Navigation Ranging).

  • TRIG (Trigger): Излучатель. Посылает ультразвук.
  • ECHO (Receiver): Приемник. Ловит отраженную волну.

09. Физика измерения

Формула расчета:

Distance = (Time * SpeedOfSound) / 2

Делим на 2, так как звук летит до стены и обратно.

10. Светодиод (LED)

Полупроводниковый прибор.

ВАЖНО: Имеет полярность.

  • Анод (Длинная нога): Плюс (+).
  • Катод (Короткая нога): Минус (-).

11. Резистор

Ограничитель тока.

Если подключить LED к 5V напрямую — он сгорит за 0.1 сек.

Мы используем резистор 220 Ом для защиты.

12. Пьезо-излучатель

Piezo Buzzer.

Преобразует электрические колебания в звук.

Управляя частотой тока, мы меняем высоту ноты.

ENGINEERING PROCESS

14. Старт проекта

1. Создайте новый проект (New Circuit).

2. Перетащите Arduino Uno.

3. Перетащите Breadboard Small.

15. Питание: Плюс

Подключаем основное питание к макетке.

5V ---> Рельса (+)

16. Питание: Земля

Замыкаем цепь.

GND ---> Рельса (-)

17. Монтаж Датчика

Найдите Ultrasonic Distance Sensor (4 пина).

Установите его на макетную плату (например, в отверстия E1, E2, E3, E4).

18. Датчик: Питание

Подключаем VCC датчика к энергии.

Sensor VCC ---> Рельса (+)

19. Датчик: Земля

Подключаем GND датчика.

Sensor GND ---> Рельса (-)

20. Датчик: Сигнал TRIG

Управление излучателем.

Sensor TRIG ---> PIN 12

21. Датчик: Сигнал ECHO

Чтение приемника.

Sensor ECHO ---> PIN 11

22. Монтаж Светодиодов

Установите 3 светодиода в ряд:

  • Зеленый
  • Желтый
  • Красный

Катоды (короткие ноги) должны смотреть в одну сторону.

23. Монтаж Резисторов

Для каждого светодиода:

Катод (Короткая нога) ---> Резистор ---> Рельса (-)

24. Подключение: Зеленый

Индикация безопасности.

Анод Зеленого ---> PIN 5

25. Подключение: Желтый

Индикация предупреждения.

Анод Желтого ---> PIN 6

26. Подключение: Красный

Индикация тревоги.

Анод Красного ---> PIN 7

27. Монтаж Динамика

Установите Piezo на плату.

Черный провод ---> Рельса (-)

28. Подключение Звука

Управление тональностью.

Красный провод ---> PIN 8

29. Проверка сборки

Чек-лист инженера:

  • [ ] Питание на рельсах есть?
  • [ ] Резисторы на минусах светодиодов стоят?
  • [ ] Датчик подключен к 11 и 12?

34. Void Setup (Свет)

35. Void Loop (Импульс)

Генерация ультразвука.

void loop() { long duration, cm; digitalWrite(trig, LOW); delayMicroseconds(2); digitalWrite(trig, HIGH); delayMicroseconds(10); digitalWrite(trig, LOW);

36. Void Loop (Расчет)

Прием эха и математика.

duration = pulseIn(echo, HIGH); // 58 - коэффициент для см cm = duration / 58; Serial.println(cm);

37. Логика (Сброс)

Гасим все перед проверкой.

digitalWrite(green, LOW); digitalWrite(yellow, LOW); digitalWrite(red, LOW); noTone(beep);

38. Условия: Зеленый

Дистанция > 100 см.

if (cm > 100) { digitalWrite(green, HIGH); }

39. Условия: Желтый

Дистанция от 30 до 100 см.

else if (cm > 30) { digitalWrite(yellow, HIGH); tone(beep, 500); }

40. Условия: Красный

Дистанция < 30 см (Критическая).

else { digitalWrite(red, HIGH); tone(beep, 1000); }

41. Финализация

Закрываем цикл.

delay(100); }

44. Отладка

Если не работает:

  • Горит все время красный? (Проверь Trig/Echo).
  • Не горит свет? (Проверь полярность диодов).
  • Нет звука? (Проверь 8 пин).
